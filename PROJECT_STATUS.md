# Thermography Website — Project Status

## Overview
A professional website for a thermography (health screening) business, built with Django + Wagtail CMS. The business owner is non-technical and needs to manage all content, availability, and pricing from an intuitive admin panel.

## Key Requirements
- **Free / low-cost hosting** — Railway or Google Cloud Run free tier
- **Professional & beautiful** interface with scroll animations (AOS)
- **Non-technical owner** must manage everything from Wagtail admin (simpler than WordPress)
- **No client login** — clients book by selecting a slot and entering contact info
- **Secure** — Django built-in security (CSRF, XSS, SQL injection protection)
- **Modular code** — each feature is a separate Django app
- **Python backend** — developer is learning Django/web dev, knows Python basics
- **Package manager**: `uv` (not pip)
- **Python version**: 3.13 (pinned via uv)
- **Future**: AI booking agent (LangGraph), payment integration (Stripe)

## Tech Stack
- **Framework**: Django 6.0 + Wagtail 7.3
- **Styling**: Tailwind CSS (CDN) + AOS scroll animations
- **Database**: SQLite (dev) → Postgres (production)
- **Package manager**: uv
- **Deployment target**: Railway (primary) or Google Cloud Run (free tier)
- **Repo**: https://github.com/Torrtheb/thermography-site

## Website Pages
| Page | Django App | Description |
|---|---|---|
| Welcome / Home | `home` | Hero, about, testimonials, news, CTA — StreamField (admin-editable) |
| Services | `services` | Service cards with descriptions, duration, pricing |
| Who We Are | `about` | About us + team members |
| Resources | `blog` | Articles, blog posts, external article links |
| FAQ | `faq` | Collapsible Q&A accordion |
| Contact | `contact` | Contact form with email notification |
| Book Appointment | `booking` | Calendar, slot picker, booking form (no login), confirmation |

## Design Decisions
- **Tailwind via CDN** (no build tools) — switch to compiled for production later
- **SQLite for dev** — switch to Postgres for production (one settings change)
- **StreamField for homepage** — owner can add/remove/reorder sections freely
- **Slide-in sidebar** on mobile (not dropdown) — main content stays visible
- **Blue brand palette** → **Teal brand palette** — matches business logo, calm healthcare feel
- **Logo in header & footer** — static file at `static/images/logo.png`
- **No client login** — booking is name/email/phone only
- **lg breakpoint** for mobile nav — prevents link text wrapping

## Common Commands
```bash
uv run python manage.py runserver          # start dev server
uv run python manage.py makemigrations     # generate migration after model change
uv run python manage.py migrate            # apply migrations to database
uv run python manage.py createsuperuser    # create admin user
uv add <package>                           # install a package
```
