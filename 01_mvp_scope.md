# Thermography Website MVP Scope (v1)

## 1) Project Goal
Launch a professional thermography website where clients can learn about services and book appointments, while the owner can intuitively edit all content in a simple admin UI. 

## 2) Target Users
- Client visitor (no login)
- Owner editor (login to admin only)

## 3) Pages Included in v1
- Home
- Services + pricing
- Meet your technician
- Articles/Blog
- News/Events
- FAQ
- Contact
- Book (external booking embed)

## 4) Owner Editable in v1
- Homepage sections and order
- Text content
- Images
- Services and prices
- Technician bio and photo
- Blog posts
- News/events entries
- FAQ items
- Contact details
- Booking link/embed URL

## 5) Out of Scope for v1 (Non-Goals)
- Custom booking backend
- Client accounts/profiles
- Payments/paywall
- Client-facing AI assistant
- Advanced analytics

## 6) v1 Success Criteria
- Owner can update all key content without code
- Client can book appointment from Book page
- Site is mobile-friendly and visually professional
- Site has basic security hardening for production

## 7) v2 Candidate Features
- Custom Python booking backend
- Owner-only AI assistant
- Optional client assistant
- Paywall or paid products/services
- Advanced analytics
